# Movie Booking Web Application - Development Setup Guide

## Project Overview
Build a full-stack movie ticket booking platform with real-time seat selection and secure payments.

## Technology Stack

### Frontend
- **Framework**: React.js with TypeScript
- **State Management**: Redux Toolkit
- **UI Library**: Material-UI (MUI)
- **Routing**: React Router v6
- **HTTP Client**: Axios
- **Real-time**: Socket.io Client

### Backend
- **Runtime**: Node.js with Express.js
- **Language**: TypeScript
- **Authentication**: JWT + bcrypt
- **Validation**: Joi/Yup
- **File Upload**: Multer

### Database
- **Primary**: PostgreSQL with Sequelize/TypeORM
- **Cache**: Redis
- **Session Store**: Redis

### Third-party Integrations
- **Payments**: Stripe/Razorpay
- **Email**: SendGrid
- **File Storage**: AWS S3 (optional)
